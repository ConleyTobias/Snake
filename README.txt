Instructions
------------
Unzip "Snake.zip"
Open the "Snake" folder in WPILib VS Code
Hit F1 & click "Simulate"
Select "Sim GUI" and click "Ok"
Drag & drop "Keyboard 0" or a controller to "Joystick[0]"
Hit Teleoperated

Open Elastic Dashboard
Go to File -> Open
Select the "ELASTIC_LAYOUT" file

Controls
--------
Click "A" on a joystick or "Z" on a keyboard
Use the D-Pad for controller or WASD for keyboard
*You may need to keep Sim tab open
